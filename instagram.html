The Challenge

Building a fully functional social media platform is a complex challenge. The goal was to replicate the core features of Instagram, including the feed algorithm, image uploading, and real-time likes.

I utilized MongoDB for the robust backend to handle user data securely, while React provided a snappy, app-like experience on the frontend. The interface mimics the native iOS feel using CSS Scroll Snapping and blurred backdrops.
